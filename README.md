# VueJs + Docker

## Project setup

```
$ npm install
$ npm install vuetify
$ vue add vuetify
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
